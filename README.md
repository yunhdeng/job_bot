# JobBot - BOSS直聘自动投递工具

[![Python Version](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

一个基于 Python 的 BOSS直聘自动投递简历工具，支持自定义过滤条件、投递限制和代理设置。

## 功能特点

- 🚀 自动登录和投递简历
- 🎯 智能过滤不符合要求的职位
- 📊 详细的投递记录和统计
- ⚡ 支持代理IP设置
- 🔒 完善的投递限制保护
- 📝 自定义打招呼语
